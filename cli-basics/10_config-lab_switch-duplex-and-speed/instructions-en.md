# Switch Duplex and Speed

## Objectives

Configure and verify Ethernet duplex and speed settings on switch interfaces to ensure optimal performance and avoid mismatches.

## Instructions

1. Check the current duplex and speed settings on switch interfaces.
2. Manually configure interfaces with the correct speed and duplex settings.
3. Configure auto-negotiation on interfaces where appropriate.
4. Identify and resolve duplex mismatches.

## Verification

- Use `show interfaces status` to view speed and duplex.
- Use `show interfaces` to check for errors caused by mismatches.
