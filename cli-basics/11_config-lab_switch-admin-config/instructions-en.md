# Switch Administrative Configuration

## Objectives

Configure administrative settings on a Cisco switch including descriptions, shutdown state, and other interface-level management options.

## Instructions

1. Add descriptive text to each interface using the `description` command.
2. Administratively disable unused interfaces.
3. Configure system uptime notifications and logging.
4. Verify the administrative state of all interfaces.

## Verification

- Use `show interfaces description` to verify interface descriptions.
- Use `show ip interface brief` to check interface status.
