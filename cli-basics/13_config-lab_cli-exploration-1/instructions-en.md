# CLI Exploration

## Objectives

Explore the Cisco IOS command-line interface using context-sensitive help, tab completion, and command history.

## Instructions

1. Use the `?` command at different privilege levels to discover available commands.
2. Use tab completion to speed up command entry.
3. Practice abbreviated commands.
4. Use keyboard shortcuts (Ctrl+A, Ctrl+E, Ctrl+U, Up Arrow, Down Arrow).
5. Use the `show history` command to review previously entered commands.

## Verification

- Confirm you can navigate between user, privileged, and configuration modes.
- Demonstrate command abbreviation and completion.
