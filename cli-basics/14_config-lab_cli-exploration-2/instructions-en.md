# Config Lab: CLI Exploration 2

## Objectives

Based on Wendell Odom's blog, author of the official CCNA guides, these labs were created with Packet Tracer and CML. Here is the link to the original lab:

- [Config Lab: CLI Exploration 2](https://www.certskills.com/clab111/)

This lab provides practice with various CLI navigation, editing, and help commands. Use the lab files to explore the Cisco IOS command-line interface and become more efficient in your configuration work.
