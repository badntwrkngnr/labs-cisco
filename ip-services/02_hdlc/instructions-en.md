# HDLC Encapsulation

## Objectives

Understand and configure High-Level Data Link Control (HDLC) as the default serial encapsulation on Cisco routers.

## Instructions

1. Verify the default encapsulation type on serial interfaces.
2. Confirm that HDLC is the default encapsulation.
3. Configure IP addresses on serial interfaces.
4. Verify end-to-end connectivity.

## Verification

- Use `show interfaces serial` to verify HDLC encapsulation.
- Use `ping` to test connectivity across the HDLC link.
