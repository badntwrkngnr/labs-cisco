# Frame Relay

## Objectives

Understand Frame Relay concepts including PVCs, DLCI numbers, and the three interface types: point-to-point, multipoint, and physical.

## Instructions

1. Configure Frame Relay encapsulation on serial interfaces.
2. Map DLCI numbers to Layer 3 addresses.
3. Verify PVC status and mappings.
4. Understand the difference between Inverse ARP and static mappings.

## Verification

- Use `show frame-relay pvc` to view PVC status.
- Use `show frame-relay map` to verify DLCI-to-IP mappings.
- Use `show interfaces serial` to check Frame Relay encapsulation.
