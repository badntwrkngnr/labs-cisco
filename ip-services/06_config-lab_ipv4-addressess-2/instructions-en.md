# IPv4 Addressing

## Objectives

Apply subnetting knowledge to configure correct IPv4 addresses on router interfaces and hosts based on given requirements.

## Instructions

1. Analyze the addressing requirements for each network segment.
2. Calculate the correct subnet mask and host addresses.
3. Configure the IP address and subnet mask on each interface.
4. Configure default gateways on hosts or routers as required.
5. Verify connectivity across all subnets.

## Verification

- Use `show ip interface brief` to verify addresses.
- Use `ping` to test connectivity to all destinations.
- Use `show ip route` to verify routes.
