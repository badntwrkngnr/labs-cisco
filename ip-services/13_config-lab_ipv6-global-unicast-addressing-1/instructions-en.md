# IPv6 Global Unicast Addressing

## Objectives

Configure static IPv6 global unicast addresses on router interfaces and hosts.

## Instructions

1. Enable IPv6 unicast routing.
2. Manually configure full IPv6 global unicast addresses on interfaces.
3. Configure link-local addresses manually or allow automatic generation.
4. Verify IPv6 routing and connectivity.

## Verification

- Use `show ipv6 interface` to verify addresses.
- Use `show ipv6 route` to verify IPv6 routes.
- Use `ping ipv6` to test end-to-end connectivity.
