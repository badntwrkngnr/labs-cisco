# IPv6 Special Addresses

## Objectives

Identify and understand IPv6 special addresses including loopback, unspecified, link-local, unique local, and multicast addresses.

## Instructions

1. Identify the different types of IPv6 addresses present on each interface.
2. Verify link-local addresses are present.
3. Identify any solicited-node multicast addresses.
4. Understand the purpose of each special address type.

## Verification

- Use `show ipv6 interface` to view all IPv6 addresses.
- Use `show ipv6 route` to identify special routes.
