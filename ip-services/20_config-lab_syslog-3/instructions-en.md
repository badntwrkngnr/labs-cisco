# Syslog Configuration

## Objectives

Configure syslog on a Cisco device to send log messages to a remote syslog server for centralized monitoring and troubleshooting.

## Instructions

1. Configure the logging level (severity) for console, buffer, and remote server.
2. Set the syslog server IP address.
3. Optionally configure logging source interface and timestamp options.
4. Generate log messages and verify they appear on the syslog server.

## Verification

- Use `show logging` to verify logging configuration.
- Check the syslog server for received messages.
