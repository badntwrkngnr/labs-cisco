# IP Routing Troubleshooting

## Objectives

Diagnose and resolve common IP routing issues including missing routes, incorrect next-hops, and interface problems.

## Instructions

1. Examine the routing table on each router.
2. Identify missing or incorrect routes.
3. Check interface status and IP configurations.
4. Verify static routes are correctly configured.
5. Use ping and traceroute to isolate the problem.

## Verification

- Use `show ip route` to check routing tables.
- Use `show ip interface brief` to check interface status.
- Use `traceroute` to identify where traffic is dropped.
