# IPv6 Static Routes

## Objectives

Configure IPv6 static routes to enable connectivity between IPv6 networks across multiple routers.

## Instructions

1. Identify the remote IPv6 networks.
2. Determine the correct next-hop IPv6 address or exit interface.
3. Configure IPv6 static routes using `ipv6 route`.
4. Verify IPv6 connectivity across the network.

## Verification

- Use `show ipv6 route` to verify static routes.
- Use `ping ipv6` and `traceroute ipv6` for testing.
