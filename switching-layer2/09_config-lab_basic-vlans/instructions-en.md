# Basic VLANs

## Objectives

Create VLANs and assign switch ports to VLANs to implement basic network segmentation.

## Instructions

1. Create VLANs 10, 20, and 30 with descriptive names.
2. Assign access ports to each VLAN.
3. Verify port assignments.
4. Confirm that devices in the same VLAN can communicate, while devices in different VLANs cannot.

## Verification

- Use `show vlan brief`.
- Use `ping` to test intra-VLAN and inter-VLAN communication.
