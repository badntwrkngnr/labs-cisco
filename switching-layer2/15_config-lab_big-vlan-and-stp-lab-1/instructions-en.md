# Big VLAN and STP Lab

## Objectives

Integrate VLAN configuration, trunking, and spanning tree in a multi-switch topology.

## Instructions

1. Create all required VLANs across the switch topology.
2. Configure trunk links between all switches.
3. Set spanning-tree priorities to control root bridge placement.
4. Verify VLAN propagation, trunk status, and STP topology.
5. Confirm end-to-end connectivity for each VLAN.

## Verification

- Use `show vlan brief`.
- Use `show interfaces trunk`.
- Use `show spanning-tree`.
